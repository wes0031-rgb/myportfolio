소개

운영 문제를 “정의 → 측정 → 개선”의 흐름으로 구조화하고, SQL로 재현 가능한 분석 결과를 만듭니다.

핵심 가치
  • 지표 정의(분모/분자, 기간, 제외조건)를 명확히 해서 “해석 가능한 KPI”로 만듭니다.
  • 분석 결과를 운영팀이 바로 실행할 수 있도록, 리포트/대시보드 형태로 전달합니다.
  • 정합성 검증(누락·중복·시계열 이상치) 프로세스를 습관화합니다.
전공/경력 기반 강점
  • 생명시스템학 기반의 시스템 관점(프로세스/제약/흐름 최적화)
  • 물류/SCM 실무에서 나온 질문을 데이터로 풀어내는 문제정의 능력
  • SQL 중심으로 빠르게 데이터셋을 구성하고 반복 분석을 자동화

경력

아래는 템플릿 예시입니다. 회사명/기간/성과 수치를 실제 내용으로 교체하세요.

SCM/물류 데이터 분석 · (회사명) 2022.01 – 2025.12 · 4년

출고 SLA·리드타임·재고 정확도·운송비 등 핵심 KPI를 정의/산출하고, 운영 회의용 리포트 자동화 및 병목 구간 원인 분석을 수행.

  • 주간 운영 리포트 자동화(쿼리 표준화 + 정합성 체크)로 리포트 작성 시간 절감
  • 피킹/패킹 생산성 지표 재정의 및 작업 구간별 리드타임 분해 분석
  • 반품/클레임 데이터 세그먼트 분석으로 개선 우선순위 제시
데이터 기반 프로세스 개선 · (프로젝트/TF) 2023 – 2025

운영 프로세스를 데이터 관점으로 모델링하고, 원인/결과 구조를 설명 가능한 형태로 시각화.

  • 리드타임 분해(입고→보관→피킹→패킹→출고)로 병목 구간 식별
  • 배송비/운송비 변동 요인 분석(권역·중량·사이즈·채널)으로 비용 최적화

SQL 숙련도, 지표 설계, 데이터 품질(정합성) 검증 중심으로 구성했습니다.
